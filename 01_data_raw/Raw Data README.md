# Raw Data

Raw datasets are not included in this repository due to file size limitations.

## Data Sources

- Divvy Bike Trip Data (Chicago bike-sharing system)
- Weather data collected via API (temperature, precipitation, wind)
- Event data manually curated for analysis

## Notes

The raw data was cleaned, transformed, and aggregated using Python.  
The final dataset used for analysis can be found in the `04_data_final/` folder.
